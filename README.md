# Vibe Cafe Finder

A lightweight web app for discovering coffee shops by:

- vibe
- minimum Google star rating
- minimum review count
- neighborhood or city text match
- maximum touristiness

## Selected initial vibes

- Cozy
- Quiet / Study-friendly
- Lively
- Artsy
- Modern
- Rustic
- Local hidden gem

## Run locally

Open `index.html` directly in your browser.
